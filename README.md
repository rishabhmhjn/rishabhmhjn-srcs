Monorepo for personal packages and configs.

## Clone

Clone with submodules so vendored dependencies (for example dotfiles plugins) are available:

```bash
git clone --recurse-submodules https://github.com/rishabhmhjn/rishabhmhjn-srcs.git
```

If you already cloned without submodules:

```bash
git submodule update --init --recursive
```
